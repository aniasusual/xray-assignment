# X-Ray SDK

X-Ray SDK for debugging multi-step pipelines with automatic trace collection and analysis.

## Installation

```bash
pip install -e .
```

## Usage

See the [main project README](../README.md) for detailed usage instructions and examples.

## Features

- Automatic trace collection for multi-step pipelines
- Context propagation across async operations
- Configurable sampling strategies
- Built-in anomaly detection
- Integration with popular frameworks
